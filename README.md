<p>O Agente "Porsche Sales Sanitizer"  é um sistema de inteligência artificial corporativo desenvolvido por mim no AI Studio da Google para normalizar dados de vendas.</p>
<p>Sua missão principal é atuar como uma camada de auditoria e higienização de dados, transformando planilhas de vendas brutas de carros Porsche com dados inconsistentes em bases de dados padronizadas, validadas matematicamente e prontas para análise de inteligência de negócios (BI).</p>
O vídeo abaixo mostra a interface gráfica do agente. O agente foi criado para receber planilhas do usuário, realizar processos de tratamento de dados e retornar uma planilha tratada.

## For Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
